# qa_portfolio_benefiti

Project Name: Benefiti.rs web application

qa_portfolio_benefiti is testing project for the web application Benefiti. The Benefiti.rs platform was initiated by specific challenges that employers face when it comes to employee benefit programs and the challenges of attracting, retaining, motivating and engaging employees. The platform is a unique "one-stop-shop" approach to creating personalized, flexible and measurable benefit programs, within the employer's easily controllable budget.

By simulating real-world scenarios and user interactions, from users and HR viewes high priority functionalities are tested including responsive designd. Additionaly, some improvements are suggested in order to increse the user experience.

Here you can find test cases, bug reports and improvement for Benefiti platform. Also, as a additon to project a folders with API Testing in Postman and SQL commands found place in here.
